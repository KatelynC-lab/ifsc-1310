Thoughts on Progressive Enhancement

Progressive Enhancement is when a website is built with the basic functionality accessible to all users and as more people use the website new functionality are add determine by what type of people are on the website. It is important in modern web development because it provides a basic functional experience to all users regardless of their browser capabilities or device.
